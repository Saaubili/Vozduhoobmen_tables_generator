# PDF в Excel

Приложение для автоматической конвертации табличных данных из PDF-документов в файлы Excel.

## Описание

Приложение позволяет выгружать экспликации различных этажей из PDF-документов в структурированный формат Excel. Программа имеет графический интерфейс и адаптивные настройки для корректной работы с документами разных форматов.

## Основные возможности

- Выбор конкретных этажей для выгрузки
- Автоматическое распознавание таблиц в PDF
- Настраиваемые параметры распознавания (расстояние склейки, заголовки таблицы и т.д.)
- Логер с отслеживанием прогресса обработки

## Установка

### Использование готового exe файла

Скачайте `main.exe` из раздела Releases и запустите его. Никакой установки Python не требуется.

### Запуск из исходного кода
